What's one situation where you might use an `alert()` to tell the user something?

It seems like you would use alerts as seldom as possible, but use them if you want to make absolutely certain the user sees the message. For instance, to make sure they want to do what the status of the page suggests they want to do.

Example: Are you sure you want to exit? You have entered information on this form that will be lost if you exit the page without saving. Example: Maybe a general warning that the content you are going to may not be suitable for the workplace.
